# 📅 Events Website

A modern and responsive events landing page built using HTML5.  
This mini project showcases an attractive event management website UI with event highlights, gallery sections, and contact information.

---

## 🚀 Features

🧭 Simple and Clean Layout  
📅 Upcoming Events Showcase  
🖼️ Photo Gallery Event Section  
🛍️ Fashion Store Event Link  
🎨 Dark Theme Design  
✨ Organized Content Structure  
📱 Beginner-Friendly Responsive Structure  

---

## 🛠️ Technologies Used

- HTML5

---

## 🎨 Sections Overview

### 🏠 Header Section
- Upcoming Events title
- Event announcement message

### 🖼️ Event Gallery Section
- Event images display
- Event descriptions
- Free entry and food details

### 🛍️ Fashion Store Section
Includes:
- Fashion Store Link
- Event information
- Promotional content

### 📞 Contact Section
- Phone number
- Email address
- Location details

---

## 💡 How to Run

Clone the repository:

```
git clone https://github.com/Sivaponnuvel/Event_Management_HTML_Mini_Project
```

Open the project folder

Run `index.html` in your browser
